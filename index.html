SwipeFaster

About the Author

A simple/nonplugin solution for using more than one finger to swipe, in means to enhance swipe navigation on mobile devices*.

*This is a beta version, working only on iPad and iPhone. Android's webkit seems to get only one finger on swipe :(

The goal here is to speed up navigation when you're looking inside a 30 items list or so. It's pure javascript, you can use with jQuery, Mootools, etc.

var mySwipe = new SwipeFaster(getElementById("slideshow"), handler); // calls the mathod passing a DOM Element (in this case, the slideshow element) and the handler for the swipe event

swipe.init() // add events to the DOM Element, calling the handler every time you swipe

swipe.kill() // clear variables and remove all events listeners

NOTE: The handler will only show you how many fingers you used, ex: -2, 3. Is up to your logic and need to find the better use for it. Nevertheless, this is kind a practical example, right? ;)
